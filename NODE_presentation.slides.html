Permutations

An ordering of elements

Consider the permutation of $1,\dots,5$:

$$ \pi = 2\ 1\ 4\ 5\ 3 $$

Maximal Increasing Subsequence (MISS)

A subsequence of increasing elements in the permutation that cannot be extended

For $2\ 1\ 4\ 5\ 3$

$1\ 4\ 5$ is a MISS corresponding to indices $2\ 3\ 4$

Working towards the modulus

$\Gamma$ is a family of objects $\gamma$.

Then $\mathcal{N}$ is the usage matrix of objects in $\Gamma$.

For example, the entry for the previous MISS would be $\begin{bmatrix} 0 & 1 & 1 & 1 & 0 \end{bmatrix}$

We define $\rho \in \mathbb{R}_{+}^{N}$ (essentially node weights) and the $\rho$-length as $\ell_\rho(\gamma) := \sum_{v=1}^{N} \mathcal{N}(\gamma, v)\rho(v)$ (the "cost" of a MISS)

And admissibility: $\rho$ is admissible for $\Gamma$ if $\ell_\rho(\gamma) \geq 1 \: \forall \: \gamma \in \Gamma$

Now we can define the p-Modulus $\text{Mod}_p(\Gamma) = \inf\limits_{\rho \in \text{Adm}(\Gamma)} \sum_{v=1}^{N} \rho(v)^p$

Two algorithms for computing modulus:

  • Basic Algorithm
  • Plus-1 Algorithm

Both of these algorithms need the shortest_path or the $\rho$-lightest MISS given some node weights

First, we generate all MISS

This is a subgraph of all up and right paths

Observation

The Basic Algorithm computes $\rho^*$ - the optimal density

The Plus-1 Algorithm computes $\eta^*$ where $\eta(v) = \mathbb{P}(v \in \text{MISS})$

$\eta^* = \frac{\rho^*}{\|\rho^*\|^2}$

$\text{Mod} = (\sum_{i \in \eta^*} \eta(i)^2)^{-1}$
